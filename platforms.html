Platform deep-dives

How migration works, platform by platform.

Every platform stores data differently. Here's exactly how ShiftItNow connects to each one and what moves across.

WooCommerce

Available now

How it connects

Connects via the WooCommerce REST API (v2/v3) using your store URL and API keys. No plugin install needed — generate Read keys in WooCommerce → Settings → Advanced → REST API and paste them in our dashboard.

What migrates

  • Products, variants, images & galleries
  • Product categories → Shopify collections
  • Custom attributes → Shopify metafields
  • Customer accounts, addresses & groups
  • Order history with line items, taxes, refunds
  • Coupons & discount rules
  • WordPress blog posts & pages
  • Yoast / RankMath SEO meta → Shopify SEO fields
  • Product reviews (via Shopify Reviews metafield)
  • 301 URL redirects auto-generated

Platform notes

  • Supports WooCommerce 3.x through 8.x
  • Subscription products require Scale plan
  • ACF / custom post type fields on Enterprise
  • Multi-currency orders preserved with original currency code

Magento / Adobe Commerce

Available now

How it connects

Uses the Magento 2 REST API natively. Magento 1.x stores connect via our lightweight bridge module (installed in minutes, removed after migration). Adobe Commerce Cloud supported with a service account token.

What migrates

  • Simple, configurable, bundled & grouped products
  • Complex attribute sets → Shopify product options
  • Tiered & customer-group pricing
  • Multi-website / multi-store catalog (Enterprise)
  • B2B customer accounts & company groups
  • Order history including invoices & credit memos
  • CMS pages, blocks & URL rewrites → redirects
  • Product URL keys preserved as Shopify handles

Platform notes

  • Magento 1.9 support via bridge module only
  • Custom EAV attributes mapped on Scale+ plans
  • Multi-store migrations need Enterprise plan
  • Downloadable products → Shopify digital files

Wix

Available now

How it connects

Authorized via Wix OAuth — click "Connect Wix" in our dashboard, approve in one screen, and we pull data through the Wix Stores API and Wix Data API. No manual exports or CSV files.

What migrates

  • Full product catalog with variants & options
  • Product images & media
  • Wix collections → Shopify collections
  • Customer accounts & order history
  • Wix Blog posts with authors & categories
  • Site pages with HTML content preserved
  • SEO title & description meta fields
  • 301 redirects from old Wix URLs

Platform notes

  • Wix Stores only (not legacy Wix eCommerce)
  • Wix Bookings & Events not included
  • Member-gated content not migrated
  • Wix restaurant menus excluded from scope

Squarespace

Available now

How it connects

Uses the Squarespace Commerce API with OAuth. Authorize once from your Squarespace account — we handle token refresh and pull all store data automatically. No XML exports required.

What migrates

  • Products & variants with pricing & inventory
  • Product images & descriptions
  • Customer accounts & purchase history
  • Orders with fulfilment status
  • Blog posts with published dates & authors
  • Static pages (About, Contact, etc.)
  • SEO meta titles & descriptions
  • Image alt text preserved

Platform notes

  • Supports Squarespace 7.0 and 7.1
  • Member Areas content not migrated
  • Donation products excluded from scope
  • Squarespace scheduling (Acuity) not included

BigCommerce

Available now

How it connects

Uses the BigCommerce Management API v2/v3 with store-level OAuth. Authorize via the BigCommerce control panel — no CSV exports or third-party connectors needed.

What migrates

  • Products, variants, categories & brands
  • Customer accounts & address books
  • Orders & fulfilment history
  • Price lists & customer groups
  • Shipping zones & custom fields
  • Blog posts & page content
  • 301 redirect rules auto-generated

Platform notes

  • Multi-channel support included by default
  • B2B Edition customer groups supported
  • Stencil theme content preserved as Shopify pages
